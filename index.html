1. Radiobutton

Один з елементів графічного інтерфейсу користувача, який дозволяє обрати один варіант з наведених. Застосовується в тестах/опитуваннях, при виборі опції в налаштуваннях тощо.

2. Checkbox

Один з елементів графічного інтерфейсу користувача (надалі не буду це дублювати), який дозволяє обрати кілька варіантів одночасно. Має ряд спільних із Radiobutton сфер застосування, але з урахуванням особливостей, зокрема в можливості перемикання декількох опцій одночасно.

Оберіть улюблені кухні з переліку:

3. Text input

Поле для введення тексту користувачем. Застосовується в полях пошуку, введення інформації в формах, при вході в акаунт чи реєстрації тощо

4. Tabs

Дозволяє перемикати видимий контент, відображаючи те, що відповідає обраному користувачем табу. Застосовується при, наприклад, відображенні одного з різних варіантів рішення.

print("Hello, World!")
console.log("Hello, World!");
Console.WriteLine("Hello, World!"); // при Top-level statements

5. Button

Дозволяють запускати дію після натискання. Типовим прикладом застосування є підтвердження чи скасування дії.

Submit покаже повідомлення про успіх, а Cancel - про скасування

6. Text label

Напис. Застосовується для пояснень та приміток до елементів GUI.

7. Link

Дозволяють переходити на інші ресурси або частини сторінки. Наприклад, класика: посилання на рікрол.

Рікрол Перейти до Tabs

8. Tooltip

Підказка, яка з'являється при наведенні на елемент. Такі підказки дозволяють зрозуміти, що треба вводити, або для чого потрібен той чи інший компонент.

9. Dropdown list

Випадаючий список, що дозволяє обрати один з варіантів. Гарно підходить, коли, наприклад, треба зекономити місце на екрані юзера. Приклад: вибір характеристик для замовлення.

10. Data grid

Таблиця, з якою користувач може взаємодіяти. Широко застосовується для зручного представлення даних. Наприклад, щоб показати товари з категоріями. Може підтримувати сортування, пошук та пагінацію.
